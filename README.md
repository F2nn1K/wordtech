# worldTech - Eletronicos e Acessorios

Site e-commerce moderno para venda de produtos de tecnologia, eletronicos e acessorios.

## Sobre o Projeto

A worldTech e uma loja online focada em oferecer os melhores produtos tech com precos competitivos e entrega para todo o Brasil. O site foi desenvolvido com foco em design moderno, experiencia do usuario e performance.

## Funcionalidades

- Layout responsivo (desktop, tablet e mobile)
- Carrinho de compras completo com persistencia no localStorage
- Filtro de produtos por categoria
- Busca de produtos em tempo real
- Efeitos 3D interativos nos cards de produto
- Particulas animadas no background
- Countdown de promocoes
- Notificacoes toast ao adicionar produtos
- Navegacao suave com scroll spy
- Formulario de newsletter

## Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Animacoes, Glassmorphism)
- JavaScript (Vanilla)
- Font Awesome (icones)
- Google Fonts (Inter)
